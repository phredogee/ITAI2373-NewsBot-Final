# NewsBot Intelligence System 2.0

## Overview
NewsBot Intelligence System 2.0 is an NLP-based news analysis platform that performs preprocessing, classification, sentiment analysis, named entity recognition, topic modeling, summarization, multilingual analysis, and conversational querying.

## Features
- Text preprocessing
- TF-IDF feature extraction
- News classification
- Sentiment analysis
- Named entity recognition
- Topic modeling
- Text summarization
- Multilingual demo
- Conversational query demo
